LogForge | Hack The Box

OS Linux December 23, 2021 Medium

LogForge es una maquina de HackTheBox basando su nombre en la tecnica de Log4Shell, vulnerabilidad de la biblioteca log4j de Java. Este ataque implica vulnerar el servicio web Apache Tomcat para obtener acceso al gestor como Administrador, partiendo de este ataque obtendremos una Shell Reversa como usuario, concatenado a esto tenemos un servicio interno de FTP que se ejecuta como Root, este servicio FTP al igual que tomcat esta basado en Java de manera que siguiendo el concepto de Log4j y apuntando a las .venv (Variables de Entorno) de Java obtendremos las credenciales de acceso como Root.
